•	Cleaned and transformed 15,172 records in Power BI, restructuring a raw 47-column dataset into a relational model with 6 linked tables for more efficient analysis.

•	Developed KPI cards, comparison bar/column charts, map-based ZIP code analysis, and interactive slicers to support analysis of pricing, property types, locations and key housing attributes.
